# Virus Genome Analysis

Genome analysis project for identifying a mystery virus found in Vero cell culture.

## Included Folders

- output/ : analysis results
- reference/ : reference viral genome files
- slurm/ : job scripts used on HPC cluster

## Excluded

- seqs/ raw sequencing reads not uploaded due to size

## Tools Used

- Centrifuge
- Minimap2
- Samtools
- Bcftools
- SLURM
- HCC Cluster
